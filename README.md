# Ego-Planner-2D-ROS2
通过对优化器底层进行修改，使其优化的轨迹坐标维度为2维（x和y），并对grid map进行重写，使其真正适配地面2D移动机器人。

# how to use ?

ROS2 compile

1.运行节点motion_plan
2.运行rviz2
3.设置好全局轨迹与障碍物信息
